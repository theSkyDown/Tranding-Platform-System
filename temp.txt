用户表 user
	获取所有用户功能（包含分页、关键字搜索功能）++
	用户添加功能(添加时无法分配角色) ++
	用户信息修改功能 ++
	用户密码重置功能 ++
	用户删除功能(admin无法被删除) ++
	用户批量删除功能(admin无法被删除) ++
	用户登陆功能 ++
	用户详细信息查看功能 +
	用户密码修改功能 ++
	用户状态切换功能(admin无法被禁用) ++
	统计用户数量功能 ++

权限(admin权限高于一切)：
	密码重置：只能重置权限等级比自己低的用户 +
	删除(包括批量)：只能删除权限等级比自己低的用户 +
	状态切换：只能切换权限等级比自己低的用户 +
	修改信息：只能修改权限等级比自己低的用户，无法将用户的角色修改成 角色的权限等级 超过 自己当前角色的权限等级 +


	字段：
		user_id int  	        用户标识
		username varchar  	用户名
		phone varchar           手机号码
		password varchar	用户密码
		gender int		性别
		role_id int		角色标识
		avatar varchar		头像
		create_time datetime	创建时间
		update_time datetime	最后更新时间
		status int		用户状态
		
角色表 role
	获取所有角色信息（分页、关键字查询）++
	角色添加功能 ++
	角色删除功能(无法删除基础角色) ++
	角色批量删除功能(无法删除基础角色) ++  
	角色修改功能 ++
	统计角色数量功能 ++

权限：
	删除(包括批量)：只能删除权限等级比自己低的角色 + 
	修改：只能修改权限等级比自己低的角色，无法将角色等级设置为超过自己的权限等级 +

	字段：
		role_id int		角色标识
		role_name varchar	角色名称
		level			角色权限等级
		create_time 		角色创建时间

	资源的删除 ++
	资源的批量删除 ++

	统计资源数量功能 ++ 
	
	字段：

权限表 permission
	角色功能限制功能 ++
	获取所有权限（分页）++
	统计所有权限的数量 ++
	权限分配功能(权限添加、删除功能) ++

权限：
	添加：无法给角色权限等级比你高的角色分配资源 +
	删除(包括批量) ：无法删除权限等级比你高的资源 +

	字段：
		permission_id int 	权限标识
		role_id int		角色id
		module_id int		资源id
